# Muaaz G. Awan — Personal Website

A simple, fast personal website built with plain HTML/CSS/JS. No build tools required.

## Files

- `index.html` — all the page content (text lives here)
- `style.css` — colors, fonts, layout
- `script.js` — mobile menu + footer year
- `assets/` — put your `profile.jpg` photo here

## Preview locally

Double-click `index.html` to open it in your browser.

## Things you'll want to edit

Open `index.html` in any text editor and look for these `<!-- comments -->`:

1. **Profile photo** — save your photo as `assets/profile.jpg`.
2. **Email** — find `YOUR_EMAIL@example.com` (appears twice) and replace it.
3. **LinkedIn** — find `YOUR_LINKEDIN` and replace with your handle.
4. **Google Scholar** — find `YOUR_SCHOLAR_ID` and replace with your profile ID.

Any other text (job descriptions, publications, etc.) is plain text in `index.html` —
just edit it directly between the tags.

## Publishing on GitHub Pages

See the chat instructions, or in short:

1. Create a GitHub repository named `YOUR_USERNAME.github.io`.
2. Upload all files from this folder into it.
3. In the repo: Settings → Pages → Source: `main` branch, `/root`.
4. Your site goes live at `https://YOUR_USERNAME.github.io`.
